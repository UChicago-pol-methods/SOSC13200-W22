Data for:
Card, D. and Krueger, A. B. (1994). Minimum Wages and Employment: A Case Study of the Fast-Food Industry in New Jersey and Pennsylvania. American Economic Review.

Original data source: http://econ-www.mit.edu/faculty/angrist/data1/mhe/card

  Obs:   820

	id           unit id
	nj           = 1 if new jersey
	d            = 1 if after nj min wage increase
	d_nj         nj*d interaction
	bk           = 1 if burger king
	kfc          = 1 if kentucky fried chicken
	roys         = 1 if roy rodgers
	wendys       = 1 if wendys
	co_owned     = 1 if company owned
	centralj     = 1 if in central nj
	southj       = 1 if in southern nj
	pa1          = 1 if in pa, northeast suburbs of philadelphia
	pa2          = 1 if in pa, easton etc
	fte          full time-equivalent employees
	ft           # full time employees
	pt           # part time employees
	mgrs         # managers/assistantt managers
	wage         starting wage ($/hr)
	meal         price of medium soda, small fries, + entree, including tax
	hrsopen      number hrs open per day
	bonus        1 if cash bounty for new workers
	ncalls       number of call-backs*
	inctime      months to usual first raise
	firstinc     usual amount of first raise ($/hr)
	nregs        number of cash registers in store

*Note: number of call-backs = 0 if contacted on first call
